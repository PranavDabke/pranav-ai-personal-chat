## README

# Twin Car Racing OpenGL Game

A 3D Twin Car Racing game developed using C++, OpenGL, and GLFW.

## Features

- Twin car gameplay
- Collision detection
- Real-time score system
- Dynamic obstacle generation
- Explosion effects
- OpenGL rendering
- GLFW controls

## Technologies Used

- C++
- OpenGL
- GLFW

## Controls

- A / D → Move Left Car
- Left / Right Arrow → Move Right Car
- ENTER → Start Game
- R → Restart
- ESC → Exit
