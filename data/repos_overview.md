## AI-Bill-Splitter
- Language: Dart (Flutter)
- URL: https://github.com/PranavDabke/AI-Bill-Splitter
- Description: AI-powered Flutter app that extracts restaurant bills using the Gemini Vision API and splits expenses fairly among groups. Cross-platform (Android + Web). Uses Google Gemini 2.5 Flash Vision API to extract structured item data from bill photos, eliminating manual entry. Implements proportional tax/tip allocation per person. Tested across 20 end-to-end test cases.
