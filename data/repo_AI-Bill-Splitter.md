# AI Bill Splitter

AI-powered Flutter application that extracts restaurant bill items using Google's Gemini Vision API and splits expenses fairly among groups.

## Features

- AI-powered receipt scanning
- Gemini Vision API integration
- Automatic bill extraction
- Multi-person bill splitting
- Shared item allocation
- Proportional tax and tip distribution
- Bill history persistence
- Dark and light theme support
- Share summary as text or image
- Cross-platform Flutter support

## Tech Stack

- Flutter
- Dart
- Gemini 2.5 Flash API
- Shared Preferences
- image_picker
- share_plus
- HTTP API integration
